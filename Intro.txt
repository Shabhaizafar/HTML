=> What is a HTML ? 
-> HTML stands for Hyper Text Markup Language.
                1. Hyper Text : 
                     i. Hyper  : Hype (Linking)  (depands on Text)
                    ii. Text   : Content (text,table,video,image..etc)

                2. Markup Language : 
                    text-encoding System.

=> HTML Programing or Not? --> Not.
-> HTML is a Markup Language.
